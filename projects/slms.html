Sales and Logistics ERP System

Updated: Sep 2019

SpringBoot RabbitMQ React Redis TypeScript Java/C#/C++

Descriptions

In this project, I developed key components of a Sales and Logistics ERP system, focusing on warehouse management and logistics settlement modules. Using technologies such as Spring Boot, Redis, Socket, WebSocket, Java, React, MySQL, and Oracle, I aimed to streamline sales and logistics operations for businesses.

Project Overview

Warehouse Module: Implemented with Spring Boot and Redis, this module automates inventory management processes, using Socket and WebSocket for real-time inventory tracking.

Logistics Settlement Module: Developed with Java, MySQL, and Oracle, it automates the calculation of transportation costs and invoicing, with a React frontend for user interactions.

Technology Stack and Implementation Details

Backend & Frontend: Leveraged Spring Boot for the backend, enhancing system efficiency with Redis caching. The React-based frontend offers an intuitive user experience.

Real-time Data: Utilized Socket and WebSocket for immediate updates in inventory and logistics, improving system responsiveness.

Database: MySQL and Oracle databases were used to store business data, including inventory information, logistics costs, and user account details, ensuring data persistence and consistency.

Database Management: Employed MySQL and Oracle for reliable data storage, ensuring smooth operations in warehouse and logistics management.

Contributions

Automated key sales and logistics processes, significantly improving operational efficiency and accuracy.

Enhanced the user interface and system interaction, facilitating better management and decision-making for businesses.
